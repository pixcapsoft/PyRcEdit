# Security Policy

## Supported Versions
We release patches for security vulnerabilities in the following versions:

| Version | Supported |
| ------- | ---------- |
| Latest  | ✅ |
| Older   | ❌ |

Only the latest release of PyRcEdit is actively supported. Please upgrade to the latest version before reporting issues.

## Reporting a Vulnerability
If you discover a security vulnerability in PyRcEdit:

1. **Do not open a public issue.**
2. Email your report privately to: pixcapsoft@gmail.com (or your preferred contact).
3. Include:
   - A clear description of the vulnerability
   - Steps to reproduce
   - Potential impact
   - Suggested fix (if possible)

We will:
- Acknowledge receipt within 48 hours
- Provide a status update within 7 days
- Work on a fix and release a patched version as soon as possible

## Responsible Disclosure
We kindly ask that you:
- Do not exploit the vulnerability
- Do not share details publicly until a fix is released
- Give us reasonable time to address the issue

Your efforts help keep PyRcEdit safe for everyone.
